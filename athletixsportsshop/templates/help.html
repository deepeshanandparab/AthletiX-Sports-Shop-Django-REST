Coupon Code
1. Valid Coupon Code

- On Cart Page, you can apply a coupon code to get discount on products.

- If valid coupon code is applied, then discounted price will be deducted from product price.

- A valid coupon code is coupon which is currently active, haven't reddemed by user previously, haven't exceed redeem count or any other specfic condition set by administrator.

2. Invalid Coupon Code

- On Cart Page, you can apply a coupon code to get discount on products.

- If invalid coupon code is applied then it will display an error.

- Enter any product related keyword in textbox next to search button above product listing section
